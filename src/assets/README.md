# Assets

`logo-gestek.svg` es una versión vectorial **placeholder** del logo real (tres figuras + engranaje + tipografía GESTEK).

Cuando tengas el PNG original que pegaste en chat:

1. Guárdalo como `logo-gestek.png` en esta carpeta (`frontend/src/assets/`).
2. En los imports, cambia `import logoUrl from './assets/logo-gestek.svg'` por `'./assets/logo-gestek.png'`.

El SVG mantiene proporciones y se ve bien en navbar (alto ~32-40px) sin pixelar.
